# 10 Open-Source AI Tools

A curated list of 10 open-source AI / developer tools, captured from the
[@aipagedaily](https://www.instagram.com/aipagedaily/) "These 10 open source AI tools" post.
Star counts (⭐) reflect the figures shown in that post.

| # | Tool | Stars | One-liner |
|---|------|-------|-----------|
| 1 | [Coolify](https://github.com/coollabsio/coolify) | 56.4K | Self-hosted Heroku/Vercel alternative |
| 2 | [OpenHands](https://github.com/All-Hands-AI/OpenHands) | 75.8K | Autonomous AI dev agent |
| 3 | [Maxun](https://github.com/getmaxun/maxun) | 15.7K | No-code web scraper |
| 4 | [Open WebUI](https://github.com/open-webui/open-webui) | 140K | Self-hosted ChatGPT-style UI |
| 5 | [Browser Use](https://github.com/browser-use/browser-use) | 83.5K | AI agent that drives your browser |
| 6 | [Langflow](https://github.com/langflow-ai/langflow) | 194K | Visual AI workflow / RAG builder |
| 7 | [Supabase](https://github.com/supabase/supabase) | 102K | Open Firebase alternative |
| 8 | [Stirling PDF](https://github.com/Stirling-Tools/Stirling-PDF) | 80K | Local all-in-one PDF toolkit |
| 9 | [Crawl4AI](https://github.com/unclecode/crawl4ai) | 67.8K | Website → clean LLM-ready data |
| 10 | [Dify](https://github.com/langgenius/dify) | 143.8K | Production AI app platform |

---

## 1. Coolify — 56.4K ⭐
[github.com/coollabsio/coolify](https://github.com/coollabsio/coolify)

- Deploy any app without touching a cloud bill
- Self-hosted Heroku and Vercel alternative
- Push to Git; it ships and runs on your own server

## 2. OpenHands — 75.8K ⭐
[github.com/All-Hands-AI/OpenHands](https://github.com/All-Hands-AI/OpenHands)

- Hand an AI agent a coding task and walk away
- It writes code, runs commands, browses the web
- The autonomous dev that actually ships

## 3. Maxun — 15.7K ⭐
[github.com/getmaxun/maxun](https://github.com/getmaxun/maxun)

- Scrape any website with zero code, just clicks
- Point, select, and it builds the scraper for you
- Turns sites into APIs and spreadsheets automatically

## 4. Open WebUI — 140K ⭐
[github.com/open-webui/open-webui](https://github.com/open-webui/open-webui)

- Run a ChatGPT-style interface on your own machine
- Plug in Ollama or any OpenAI-compatible model
- Works fully offline; zero data leaves your box

## 5. Browser Use — 83.5K ⭐
[github.com/browser-use/browser-use](https://github.com/browser-use/browser-use)

- Tell an AI agent to use any website for you
- Fills forms, shops, books, scrapes on command
- Your computer finally does the boring stuff itself

## 6. Langflow — 194K ⭐
[github.com/langflow-ai/langflow](https://github.com/langflow-ai/langflow)

- Drag and drop AI workflows, no code needed
- Build agents and RAG pipelines visually
- Export the whole thing as a Python API

## 7. Supabase — 102K ⭐
[github.com/supabase/supabase](https://github.com/supabase/supabase)

- Spin up a Postgres database in seconds
- Get auth, storage, and realtime APIs free
- The open Firebase alternative everyone switched to

## 8. Stirling PDF — 80K ⭐
[github.com/Stirling-Tools/Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF)

- Merge, split, sign, and OCR PDFs locally
- 50 plus tools, runs entirely on your server
- Nothing ever uploads to a random website

## 9. Crawl4AI — 67.8K ⭐
[github.com/unclecode/crawl4ai](https://github.com/unclecode/crawl4ai)

- Turn any website into clean data for your LLM
- Outputs ready to feed straight into AI models
- Handles JavaScript-heavy pages most scrapers choke on

## 10. Dify — 143.8K ⭐
[github.com/langgenius/dify](https://github.com/langgenius/dify)

- Ship production AI apps without glue code
- Combines agents, workflows, and RAG in one platform
- Self host it, own every byte of your data
